Designed to track and log various changes and activities that occur on a WordPress site.

== Description ==
Post Creation and Update: Logs the creation and updating of posts, recording the post type, action (created/updated), post title and ID, status, and the user ID.
Post Deletion: Captures and logs the deletion of posts, including the post type and status at the time of deletion.
Plugin Activation: Logs when a plugin is activated, noting the action, plugin name, and the user who activated it.
Plugin Deactivation: Similar to activation, this logs the deactivation of plugins.
Admin Page for Viewing Logs: The plugin adds a page to the WordPress admin area, displaying a log table with entries for each recorded action. This page is accessible only to administrators.
CSV Export Functionality: Administrators can export the log entries as a CSV file, allowing for external analysis and record-keeping.
Enhanced Security and Access Control: Access to the log page is restricted to administrators, ensuring sensitive information is protected.

== Installation ==
1. Upload \"activity-tracker\" folder to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
